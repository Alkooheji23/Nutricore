### **Memory API (Pseudocode)**

```js
async function getUserMemory(userId) {
  return db.query("SELECT * FROM User_Profile WHERE user_id = $1", [userId]);
}

async function saveUserMemory(userId, updates) {
  // updates = { key: value }
  for(key in updates) {
    db.query(`UPDATE User_Profile SET ${key} = $1 WHERE user_id = $2`, [updates[key], userId]);
  }
}
```

## MODULE 2: **Fitness Logic Engine**

Handles training volume, progression, RPE, deloads, and adjustments.

### **Core Concepts**

* **Volume** = sets/week per muscle group
* **Progression Rule:**

  * If user hits RPE 6–7 → increase load or reps next week
  * If RPE 8–9 → maintain
  * If RPE ≥ 9.5 or soreness high → decrease by 10–20%
* **Workout Structure:**

  * Warm-up
  * Primary lifts
  * Secondary lifts
  * Accessories
  * Finisher (optional)

### **Volume Table (example)**

| Experience   | Chest | Back  | Legs  | Shoulders | Arms |
| ------------ | ----- | ----- | ----- | --------- | ---- |
| Beginner     | 8–10  | 8–12  | 10–12 | 6–8       | 4–6  |
| Intermediate | 10–16 | 12–18 | 12–20 | 8–12      | 8–10 |

### **Progression Pseudocode**

```js
function adjustVolume(currentVolume, soreness, rpeAvg) {
  if (soreness > 7) return currentVolume * 0.8;
  if (rpeAvg > 9) return currentVolume * 0.9;
  if (rpeAvg < 7) return currentVolume * 1.1;
  return currentVolume;
}
```

## MODULE 3: **Nutrition Engine**

### **Calorie Formula**

Use Mifflin-St Jeor + activity multiplier.

```js
function calculateCalories(user) {
  const { weight, height, age, gender, goal } = user;

  const bmr = gender === "male"
    ? 10*weight + 6.25*height - 5*age + 5
    : 10*weight + 6.25*height - 5*age - 161;

  let calories = bmr * user.activityLevel;

  if (goal === "fat_loss") calories -= 300;
  if (goal === "muscle_gain") calories += 250;

  return calories;
}
```

### **Macros**

* Protein: 1.8–2.2 g per kg
* Fat: 20–30% of calories
* Carbs: remainder

### **Cultural Food Mapping (GCC Support)**

Example:

* Chicken machboos → 850 kcal
* Shawarma → 420 kcal
* Luqaimat → 150 kcal (2 pieces)
