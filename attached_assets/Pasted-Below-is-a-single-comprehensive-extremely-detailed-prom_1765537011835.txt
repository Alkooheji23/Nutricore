F. CHAT INTERFACE — COMPLETE PREMIUM REDESIGN (MANDATORY)

This is the most important part.
The current chat looks generic. It must transform into a luxury AI companion, not “a chat widget.”

1. Chat Container
	•	Background: #0A0A0A
	•	Max width: 750px
	•	Centered
	•	Large vertical spacing
	•	Invisible edges (no border)

2. Message Bubbles

User messages:
	•	Background: #1A1A1A
	•	Rounded corners: 18–24px
	•	Text: Soft white
	•	Subtle olive glow on the outer edge (very faint)

AI messages (“My Trainer”):
	•	Background: #111
	•	Rounded: 18–24px
	•	A soft golden highlight line at the top edge (2–3px), extremely subtle
	•	Text: #F2F2F2
	•	A small pulse-mountain icon at the start of the conversation (not every message)

3. Chat Input Bar

This must feel premium and futuristic.
	•	Full-width rounded pill (height: ~56px)
	•	Background: #1A1A1A
	•	Inner shadow for depth
	•	Olive or gold glowing send button (round, floating style)
	•	No harsh outlines

4. Typing Indicator

Replace the default dots with:
	•	A soft gold pulsing line (heartbeat-style), animated gently
	•	Slow pulsation (luxury avoids fast/cheap animations)

5. Scroll + Transitions
	•	Smooth scroll
	•	Messages fade in gently
	•	No jumpy movement
	•	Long conversations feel like a continuous flow

6. Header in Chat Screen

Show a minimal, elegant header:
	•	Pulse-mountain icon (small)
	•	Text: My Trainer
	•	No photo, no square shapes

Imagery must be symbolic, not literal.
